Logisim için basit bir çözüm

Logisim

  1. Logisim uygulamasına sağ tıklayıp paket içeriğini göster'e tıklayın.
  2. Contents/Resources klasörüne ilerleyip logisim.jar dosyasını içeren "Java" klasörünü alın. Uygulamanın geri kalanına ihtiyacınız yok.
  3. Logisim uygulamasını indirmediyseniz logisim.jar dosyasına buradan ulaşabilirsiniz.

Automator

  1. Automator uygulasını açın.

  2. Karşınıza çıkan ekrandan uygulama seçeneğini seçin.

  3. Arama kısmına apple yazıp AppleScript'i Çalıştır'ı fare ile ekrana sürükleyin.

  4. Aşağıdaki kodu yapıştırın ve return input satırını silin.

    set p to POSIX path of (path to me)
    do shell script "java -jar " & p & "/Contents/Java/logisim.jar"

  5. cmd ve s tuşlarına basıp uygulamanıza isim verip kaydedin. Uzantısının .app olmasına dikkat edin

    Automator'u kapatabilirsiniz.

  6. Uygulamamız klasöre kaydedildi

Son dokunuş

  1. Sağ tıklayıp paket içeriğini göster'e tıklayın.

  2. Contents klasörüne girip logisimden aldığınız Java klasörünü koyun.

    logisim.jar dosyasını bu siteden indirdiyseniz Java isimli yeni klasör oluşturun ve logisim.jar dosyasını klasörün içine taşıyın

  3. Uygulamamız kullanıma hazır.
